'Coding Dojo: Lua with LÖVE'

Sat 02 March 2013
By luch

In Events.

tags: dojolua

Lua is a lightweight and extensible language that is designed to be embedded for scripting. Which is why it's very popular in game dev. This language is a good and simple example of how to do a lot when you have a little.

LÖVE is a spiffy framework for 2D games in Lua. It's cross-platform, performant, completely free, and great for small games and prototyping.

We'll be using LÖVE to animate an abstract scene, complete with music and special effects, in a Coding Dojo format.

To get a feel for both the language and the framework, check out these links:

  • Download Lua
  • Introductory topics in the Lua Tutorial Directory
  • LÖVE Wiki (explains how to start and provides a bunch of tutorials)
